Contact Button


=== Manually ===

1. Upload the 'Contact Button' folder to the '/wp-content/plugins/' directory
2. Activate the Hotline Phone Ring plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.

== Installation ==
1. Visit \Plugins > Add New
2. Search for \Contact Button\
3. Activate Contact Button from your Plugins page.
